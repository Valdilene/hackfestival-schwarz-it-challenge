# hackfestival-schwarz-it-challenge

## Getting Started



### Start using Docker

```
docker-compose -f docker/docker-compose.yml up
```

## Native Install

### Ubuntu
```
sudo apt-get install libpq-dev
```

### MacOS

```
brew install postgresql
```

### Install Packages

```
python3 -m venv .venv
. .venv/bin/activate
pip install -r backend/requirements.yml
```
